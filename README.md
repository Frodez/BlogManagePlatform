# BlogManagePlatform  
## 这是一个springboot项目的脚手架,追求简洁高速可扩展。  
已完成了一些常用功能，包括:
1. 较简单的请求限流。
2. 较简单的日志配置。
3. 多功能的参数验证。
4. 较完备的spring security配置,使用jwt,无session。
5. 高性能的常用工具类，如对象复制，反射，json处理，日期处理，正则表达式处理，判空。
6. 具备良好API的通用Result。
7. 较全面的mybatis代码自动生成工具。
8. swagger支持。
9. 异常兜底处理。  
**对可能有不同应用场景的功能，均保证了尽可能良好的扩展性，方便接入不同实现。**
